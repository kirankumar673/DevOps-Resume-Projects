AWS NETWORK ARCHITECTURE
=========================

VPC: 10.0.0.0/16
├── Public Subnets (10.0.1/2.0/24) — AZ a, b
│     ├── Application Load Balancer (HTTPS entry point)
│     ├── NAT Gateway (outbound for private subnets)
│     └── Jenkins EC2 (CI server)
│
└── Private Subnets (10.0.3/4.0/24) — AZ a, b
      └── EKS Worker Nodes (t3.medium × 2, min 1, max 4)

SECURITY GROUPS
===============
alb-sg        → 80, 443 from 0.0.0.0/0
eks-nodes-sg  → all from alb-sg + self (node-to-node)
jenkins-sg    → 8080, 22 from admin IP only

TERRAFORM MODULES
=================
terraform/
├── main.tf           ← root (wires all modules)
├── variables.tf      ← all typed variables
├── outputs.tf        ← cluster/ALB/VPC outputs
├── terraform.tfvars  ← environment values
├── vpc/main.tf       ← VPC + subnets + IGW + NAT + route tables
├── security-groups/  ← ALB + EKS nodes + Jenkins SGs
├── eks/main.tf       ← IAM roles + EKS cluster + node group
└── alb/main.tf       ← ALB + target group + listener
